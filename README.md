 # Student Results Management System

This project is a lightweight web-based student results management system built in plain HTML, CSS, and JavaScript.

## Features
- Admin login
- Student registration
- Results entry for Mathematics, English, Science, GS, and BAM
- Division, grade, and position calculation
- SMS result notification workflow for parents
- Local persistence with browser storage

## How to run
1. Open the folder in a browser, or run the included launcher.
2. Use the default admin credentials:
   - Username: admin
   - Password: 1234

## Launcher
- On Windows, double-click run.bat
- Or run `python -m http.server 3000` from this folder
